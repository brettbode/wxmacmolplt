MacMolPlt

by Brett Bode

 

Key Features:

  • Input using:
    • MolPlt file format punched out by GAMESS
    • by pasting in Cartesian coordinates from any GAMESS or Gaussian log file
    • any GAMESS input, log or irc file
    • Xmol style XYZ file
    • MolDen format files
    • Protein Data Bank (pdb) file
    • Chemical Markup Language (CML) files
    • Build graphically with an interactive molecule editor
    • Build by hand using cartesian or Z-Matrix style internal coordinates
  • 3D display of structure and Normal mode
  • Animation of Normal modes.
  • 3D real-time rotation and translation under mouse control
  • Calculation of any Z-matrix parameter
  • Visually measure/display distances, angles and dihedrals.
  • Automated creation of Linear Least Motion (LLM) paths
  • 2D and 3D calculation and display of:
    • Molecular orbitals (AO's, MO's, or LMO's)
    • Total Electron Density
    • Density differences
    • Molecular Electrostatic Potentials (MEPMaps)
  • Basic input file generator for GAMESS
  • OpenGL used for high-quality 3D rendering
  • Support for molecular symmetry
    • Generate symmetry dependant atoms
    • Determine symmetry unique atoms
    • Automatically determine the list of supported point groups for a set of coordinates.
    • Rotate coordinates into the proper principle orientation and symmetrize the coordinates
    • Display symmetry operators for many point groups
    • Edit while enforcing symmetry
  • Publication quality output (color or black and white) via direct printing or Copy and Paste into another Application.
  • Output in a variety of forms including Windows bitmap, JPEG, GIF, PNG, CML, XMol XYZ, MDL molfile, animated GIF, Flash Movies and QuickTime movies.

Distribution:

MacMolPlt is FreeWare released under the terms of v2 of the GPL (included in the top level of the source distribution).

Bugs, questions and enhancement requests can be reported via the Issues Entry Tab or via the user survey form at the web form. In either case please include information on the version of MacMolPlt and of your OS. You can also send me email at: MMP.Guy@gmail.com
Please let me know what you like and what you don't. Your comments help drive future enhancements to the program!

NOTE: I am asking only to get an idea of who uses my program and what they like about it. I won't pester you with ads or anything like that, thus you have absolutely no excuse for not dropping me a line . . .

Requirements for Linux systems:

OpenGL
wxWidgets install with OpenGL enabled. (either from a vendor supplied library or source install).

Requirements for MacOS X:

10.4 and newer for the 32 bit universal build. MacMolPlt should run on any system MacOS X runs on.
10.7 and newer for the 64 bit version.

Requirements for Windows:

The 32 bit version is built on Windows 7, but is believed to work correctly on older Windows versions such as XP.
The 64 bit version is built on Windows 8, but should work on all 64 bit Windows OS's
